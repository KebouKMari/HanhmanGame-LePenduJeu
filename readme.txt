               ***************************************Game Hangman - For Beginners - language C *************************************

Hi friends !!!!

This is the source code of the game the Hangman (For those who don't know this game, google it :) !)

So for beginners in C, it's a great program to test your knowledge about functions, pointers, files 
and dynamic memory allocation. 
Understand the principle of the game and the code will allow you to see my implementation logic Hihi!!! and help you code your own.

Attention: When you type a letter for the first time in the game, type it again if it is not displayed.
   Example: Enter a character: a
 	  You have 5 moves left: *******
	  Type the 'a' a second time.
          Enter a character: a
	  You have 5 strokes left: ***a***

NB: Do this when you enter the very first letter, to make sure that the letter is not actually in the word.

In this game you can choose the level of the game which decreases the number of possible tries.
Example: Level 1: 15 possible trials
	 Level 2: 10 possible trials
         Level 3 : 5 number of possible trials

This program picks the mystery words from a file called "livre.txt" which contains 15 words but you can add them.

This program is not perfect and can be improved, I hope that it will help you who begin in programming like me :)
